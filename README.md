# 📊 Calculadora de Média de Alunos

Aplicação web simples desenvolvida com **HTML, CSS e JavaScript** que permite cadastrar alunos, inserir suas notas e calcular automaticamente a média ponderada.

## 🚀 Funcionalidades

- Inserir **nome do aluno** <br>
- Informar **4 notas** <br>
- Cálculo automático da **média ponderada** <br>
- Adicionar **vários alunos** <br>
- Exibição de **nome, notas e média** <br>
- Interface simples e organizada <br>

➡️ [Clique aqui para testar](https://figueredolp.github.io/)

## 🧮 Regra de cálculo

A média é calculada usando pesos diferentes para cada nota:

| Nota   | Peso |
| ------ | ---- |
| Nota 1 | 0.2  |
| Nota 2 | 0.3  |
| Nota 3 | 0.2  |
| Nota 4 | 0.3  |

Fórmula:

media = (n1 × 0.2) + (n2 × 0.3) + (n3 × 0.2) + (n4 × 0.3)

1. Digite o **nome do aluno** <br>
2. Insira as **4 notas** <br>
3. Clique em **Enviar** <br>
4. O aluno será adicionado à lista com sua **média calculada** <br>
